Игра Пинг-понг на два игрока
Для начала игры запустите файл game.py
Чтобы начать игру, подтвердите готовность каждого из игроков при помощи соответствующих кнопок
Игроки могут управлять своими 'рокетками' при помощи клавиш wasd и стрелок. 
Цель игры - запустить шарик таким образом, чтобы противник не смог его отбить,
 и шар коснулся противоположному к вам краю поля
(Функции для разработки:кнопка q позволяет на некоторое время заморозить игру, кнопка x позволяет выйти из игры.)
